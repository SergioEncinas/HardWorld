Reviews / Placas Base / Asus Prime A520M-K


     






ASUS presentó en agosto de 2020 esta placa base que consta de un zócalo para procesadores AMD (listo para las series 5000 / 4000 G / 3000 de Ryzen), refrigeración completa: disipador de calor PCH y Fan Xpert, conectividad ultrarrápida: compatibilidad con M.2, Ethernet de 1 Gb, USB 3.2 y múltiples copias de seguridad del hardware para una protección integral del sistema

Pros:
✔️ Muy buen precio
✔️ Tiene todo lo necesario para un PC de gama media
✔️Se puede conectar un SSD M.2 que siempre viene bien para que el sistema arranque con mayor velocidad
✔️Sonido Realtek de alta definición
✔️Buena refrigeración


Contras:
❌Al ser micro ATX no podemos meter tarjetas gráficas de gran tamaño
❌No permite overclock a la CPU
❌Viene con la BIOS desactualizada, hay que montar un procesador de primera generación (es decir un Ryzen 3 1200 por ejemplo valdría) para actualizarla y que admita los de generaciones superiores.
